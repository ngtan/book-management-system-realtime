## Book Management System Realtime

- PHP & MySQL
- Ratchet WebSockets for PHP
- jQuery

## Import database
- Rename book-managament-system-realtime.md to book-managament-system-realtime.sql to import into your database

## How to run

`
  $ composer update
`

`
  $ php bin/server.php
`
